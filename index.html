Maps of Value

Six transparent pastel bubbles in a row. Each bubble holds one value title and its quotes in black for maximum clarity. Use one finger to rotate, two fingers to zoom, and the AR button to place it in real space.

The page is set up for model.glb in the same GitHub folder as index.html. The AR scale is fixed so it should appear much smaller and easier to place than before.
If the model still feels too large in AR, change the model scale in your 3D file rather than pinch scaling. This version has already been rebuilt as a lighter, simpler asset for easier placement.
